# Golden calls

Keep your best end-to-end call recordings + quote JSONs here, frozen.

Why: (1) reproducible evidence for judges — the harness runs over these and the
numbers don't change; (2) regression set — after any prompt tweak, re-run the
same scenario and confirm the agent still moves price and passes honesty checks.

Minimum for submission: one call per persona (aggressive / evasive / honest),
including at least one with leverage-attributed price movement.
